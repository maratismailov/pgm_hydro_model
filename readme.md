## Теория

### Корреляция и регрессия

**Линейная регрессия** — уравнение вида `ỹ = b0 + b1·x`, предсказывающее значение `y` по известному `x`. В данном случае предсказывается температура воздуха `T` по высоте над уровнем моря `Z`.

**Коэффициент корреляции** `r` принимает значения от −1 до +1 и показывает тесноту линейной связи между переменными:

| \|r\| | Характер связи |
|---|---|
| 0.7 – 1.0 | Сильная |
| 0.4 – 0.7 | Умеренная |
| 0.0 – 0.4 | Слабая |

Знак `r` указывает направление: отрицательный — обратная связь (x растёт, y убывает), положительный — прямая.

---

### Параметры таблицы ЛИНЕЙН

| Параметр | Описание |
|---|---|
| `b1` | Коэффициент наклона. Показывает изменение `T` при увеличении `Z` на 1 км. В примере b1 ≈ −5.14°C/км |
| `b0` | Свободный член. Теоретическое значение `T` при `Z = 0` |
| `Sb1` | Стандартная ошибка коэффициента `b1` |
| `Sb0` | Стандартная ошибка коэффициента `b0` |
| `r²` | Коэффициент детерминации. Доля дисперсии `y`, объяснённая моделью. `r² = 0.70` означает, что модель объясняет 70% разброса температур |
| `S²` | Дисперсия остатков. Средний квадрат расхождения между фактическими и расчётными значениями |
| `F` | Критерий Фишера. Проверяет статистическую значимость регрессии в целом |
| `k2` | Число степеней свободы остатков. Для парной регрессии: `k2 = n − 2` |
| `Σ1` | Объяснённая сумма квадратов |
| `Σ2` | Остаточная сумма квадратов |

> `Σ1 + Σ2 = Σ_общая`. Чем больше доля `Σ1`, тем лучше модель описывает данные.

---

### Парная и множественная регрессия

**Парная** регрессия использует один предиктор:
```
ỹ = b0 + b1·Z
```

**Множественная** регрессия использует несколько предикторов:
```
ỹ = b0 + b1·Z + b2·φ + b3·λ
```

По результатам работы 2: добавление широты `φ` увеличивает R² с 0.699 до 0.710, добавление долготы `λ` практически не меняет результат (R² = 0.710). **Вывод:** высота является главным и достаточным предиктором температуры для данного набора станций.

---

### Ложная корреляция

Высокое значение `r` не означает причинно-следственную связь. Основные источники ложной корреляции:

**1. Влияние третьей переменной.**  
`x` и `y` оба зависят от скрытой переменной `z`, из-за чего кажутся связанными между собой. Пример: число потребляемых лекарств и число смертей коррелируют, поскольку оба показателя растут с численностью населения.

**2. Нелинейное преобразование переменных.**  
Если по осям отложить `y` и `y/x`, корреляция возникает автоматически — даже при отсутствии реальной связи между исходными переменными.

**3. Неравномерное распределение точек.**  
Корреляция возникает случайно из-за пространственного расположения данных. Пример из учебника: `r(φ, λ) = 0.54` для МС Кыргызстана объясняется не физической связью, а характером географического распределения станций.

> **Правило:** установленная статистическая зависимость должна получить содержательное объяснение, опирающееся на физику или логику изучаемого явления.

---

### Метод наименьших квадратов (МНК)

Коэффициенты `b0` и `b1` находятся из условия минимума суммы квадратов отклонений фактических значений от расчётных:

```
b1 = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / Σ[(xᵢ - x̄)²]
b0 = ȳ - b1 · x̄
```

Для множественной регрессии та же задача решается в матричной форме:
```
β = (XᵀX)⁻¹ Xᵀy
```

где `X` — матрица предикторов с добавленным столбцом единиц для свободного члена.

При вычислении дисперсии остатков `S²` знаменатель равен `n − k − 1`, где `k` — число предикторов. Для парной регрессии это `n − 2`. Деление на `n` вместо `n − k − 1` занижало бы оценку ошибки.

---

### `scipy.stats.linregress` — реализация МНК для парной регрессии

```python
from scipy import stats
b1, b0, r, p_value, sb1 = stats.linregress(x, y)
```

Функция принимает два одномерных массива и возвращает пять значений:

| Возвращаемое значение | Параметр | Описание |
|---|---|---|
| `slope` | `b1` | Наклон прямой |
| `intercept` | `b0` | Свободный член |
| `rvalue` | `r` | Коэффициент корреляции Пирсона |
| `pvalue` | `p` | p-значение для проверки гипотезы `b1 = 0` |
| `stderr` | `Sb1` | Стандартная ошибка наклона |

Функция **не возвращает** напрямую: `r²`, `Sb0`, `F`, `S²`, `Σ1`, `Σ2` — эти величины вычисляются отдельно на основе возвращённых значений.

**Альтернативный синтаксис** — обращение к полям результата:
```python
result = stats.linregress(x, y)
result.slope      # b1
result.intercept  # b0
result.rvalue     # r
result.pvalue     # p-value
result.stderr     # Sb1
```

**Ограничение:** функция работает только для парной регрессии (один предиктор). Для множественной регрессии используется `numpy.linalg`:

```python
import numpy as np

X = np.column_stack([np.ones(n), Z, phi, lam])  # матрица предикторов с константой
beta = np.linalg.inv(X.T @ X) @ X.T @ y         # МНК в матричной форме
```

**Порядок вычислений внутри `linregress`:**

```
1. x̄ = mean(x),  ȳ = mean(y)
2. b1 = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / Σ[(xᵢ - x̄)²]
3. b0 = ȳ - b1 · x̄
4. r  = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / √(Σ(xᵢ-x̄)² · Σ(yᵢ-ȳ)²)
5. S² = Σ(yᵢ - ŷᵢ)² / (n - 2)
6. Sb1 = √(S² / Σ(xᵢ - x̄)²)
7. p-value — из t-распределения с (n-2) степенями свободы
```

**p-value** проверяет нулевую гипотезу H₀: `b1 = 0` (зависимость отсутствует). При `p < 0.05` нулевая гипотеза отвергается — зависимость признаётся статистически значимой.
